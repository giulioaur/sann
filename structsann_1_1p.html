Detailed Description

The parameters object. It holds all the settable hyper-parameters:

  • max_epoch: The maximum number of epoch on which train.
  • mb : The mini_batch size (1 = online, inputs_size = stochastic)
  • eta : The learning rate.
  • mi : The momentum term.
  • lambda : The L2 term.
  • update(struct p &par, const size_t epoch) : This function is called every epoch, and the argument are the struct on which it is called and the epoch. This attribute allows to change the hyperparameters every epoch.
